# Refract Cards

Refract's card styling as a standalone plugin, so cards look like the
[Refract theme](https://github.com/ordureconnoisseur/stash-refract) while the
rest of your Stash stays on the default theme.

All cards get the base Refract look (glass surface, hover glow, 3D tilt).
On top of that:

- **Performer cards** — the playing-card layout: name banner with tier
  glow, diagonal tier ribbon, neon stat strip (rating, age, scenes, O
  count, country), and floating hearts on favourites.
- **Scene cards** — the classic Refract card with the Bronze → Silver →
  Gold → Diamond → Legendary → Perfect tier frame on rated cards.
- **Everything else** — galleries, images, studios, tags, groups, markers
  get the base glass/glow/tilt styling.

## Settings

Settings → Plugins → **Refract Cards** — three on/off switches:

| Switch | What it styles |
| --- | --- |
| **Performer cards** | Playing-card layout |
| **Scene cards** | Classic tier-frame card |
| **Everything else** | Base styling on all other card types |

Off = the default Stash card for that group. Saved per browser, applied
instantly.

> Don't run this alongside the full Refract theme — Refract already includes
> all of this and the two would fight over the same cards.
